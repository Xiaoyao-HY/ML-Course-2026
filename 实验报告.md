# 实验二：概率建模、KNN分类与梯度下降优化 - 实验报告

## 一、参数估计（MLE 与 MAP）

### 1.1 实验结果
- **数据**: [1, 1, 0, 1, 0]（5个样本，3个1，2个0）
- **MLE**: p = 0.6
- **MAP**: p ≈ 0.571（使用Beta(2,2)先验）

### 1.2 思考题回答

**Q1: 先验对后验的影响体现在哪里？**
- 先验（Prior）代表在没有观测数据前对参数的信念
- 后验（Posterior）是似然函数与先验的乘积（归一化后）
- 从图像可以看出：后验曲线位于似然和先验之间，相当于对两者的"折中"

**Q2: MLE 与 MAP 的位置有何差异？**
- MLE = 0.6：完全由数据决定
- MAP ≈ 0.571：向先验均值0.5收缩
- MAP 总是比 MLE 更靠近先验均值（这里是0.5）

**Q3: 当样本数量增加时，这种差异如何变化？**
- 样本少时：先验影响大，MLE与MAP差异明显
- 样本多时：数据占主导，MLE与MAP趋于一致
- 这是因为大数据会"压倒"先验的影响

## 二、KNN 分类

### 2.1 实验结果
- K=1：边界非常复杂、曲折
- K=3：边界较为平滑
- K=5：边界更加平滑，决策更"保守"

### 2.2 思考题回答

**Q1: K 较小时，模型复杂度如何变化？**
- K=1时复杂度最高，每个点都被孤立考虑
- 边界非常曲折，容易过拟合

**Q2: K 增大后，边界呈现什么变化趋势？**
- 边界越来越平滑
- 决策更"保守"，考虑更多邻居的意见

**Q3: 如何理解过拟合与泛化能力之间的关系？**
- K过小 → 过拟合 → 训练集表现好，测试集可能差
- K过大 → 欠拟合 → 模型太简单，可能丢失重要模式
- 合适的K值需要在复杂度和泛化能力间取得平衡

## 三、梯度下降法

### 3.1 实验结果
- 目标函数：f(x) = x² + 2x + 1 = (x+1)²
- 最优解：x = -1，f(x) = 0
- 初始点：x₀ = 5
- 收敛后：x ≈ -0.93（接近最优值）

### 3.2 思考题回答

**Q1: 损失函数是否单调下降？**
- 是的，在合适的学习率下损失函数单调递减
- 梯度下降保证每一步都向函数值减小的方向移动

**Q2: 参数是如何逐步逼近最优值的？**
- 从初始点x=5开始
- 沿着梯度负方向迭代更新
- 逐步从5 → 2.6 → 1.04 → 0.224 → ... → -0.93

**Q3: 学习率变化会对收敛过程产生什么影响？**
- 学习率太小：收敛慢，需要更多迭代
- 学习率太大：可能跳过最优解，甚至发散
- 合适的学习率能快速收敛且稳定

## 五、实验总结

本次实验验证了三个核心概念：
1. **概率建模**：MLE只依赖数据，MAP引入先验进行正则化
2. **KNN分类**：K值控制模型复杂度，平衡过拟合与欠拟合
3. **梯度下降**：通过迭代优化逐步逼近最优解
